### 🚀 About Me

I'm a full-stack developer and digital artist passionate about building **local-first AI tools**, **developer CLIs**, and **retro-futuristic artwork**. I focus on creating autonomous agent architectures, multi-provider LLM routers, and high-performance TypeScript tooling.

* 🏛️ Creator of **[Daedalus](https://github.com/bgill55/daedalus)** (`daedalus-cli`) — Local-first AI coding assistant with embedded model router & multi-agent orchestration.
* ⚡ Shipped **[Daedalus-Lite](https://bgill55.github.io/daedalus-lite/)** — Zero-dependency starter template to build and sell your own branded AI CLI.
* 🎨 Creative Mind: Combining retro-futuristic digital art with cutting-edge software engineering.
* ✉️ Contact: `bgill55_dev@voxvivid.com`
